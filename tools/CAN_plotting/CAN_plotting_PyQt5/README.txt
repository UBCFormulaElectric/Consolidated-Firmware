# Instructions from graphing CAN Signals

#### 1. Download and open CANvas (stable v3.6.3) from https://canlogger.csselectronics.com/downloads.php?q=canvas 

#### 2. Click on the "Convert Data" button

#### 3. Click on the DBC folder icon and select all tick boxes to add (name,int value,...etc), then click apply

#### 4. go to the "load log files" tab on the left, and enter the log file to convert.

#### 5. In the "manage log files" output tab, click the green start button.

#### 6. In options you can find where the processed log file is output.

#### 7. Run the plottingUi python script.
